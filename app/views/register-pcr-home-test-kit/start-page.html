Check if this service is right for you

Test kit registration is how the NHS matches your contact details to the COVID-19 test kit you used.

You need to register the test kit so you can get the test result.

Who should use this service

Use this service to register a PCR home test kit. PCR is the test for COVID-19 that is sent to a lab.

Who should not use this service

You need to use a different service if:

  • you went to a test site and you need to register a test kit you used at the test site
  • you travelled to the UK, and you need to register a PCR test you're doing after you entered the UK
  • you did a rapid lateral flow test yourself at home or work, and you need to report your test result to the NHS

What you need

Before you start, you will need:

  • the unique test kit barcode
  • your order ID from the confirmation email (only if you ordered your test kit and you have an order ID)
  • to find your nearest Royal Mail priority postbox, or book a courier (only if you ordered your test kit and will return it by post)

f you are registering a test for more than one person, you need to register each test to get all the results.

Get help to register a test kit

If you cannot use the online service, call 119 (free from mobiles and landlines). Lines are open every day, 7am to 11pm. 119 provides support in 200 languages.

InterpreterNow is a free online British Sign Language interpreter service for 119.

You can use the free Be My Eyes app to get help from trained NHS Test and Trace staff. Download the app, go to 'Specialized Help' and select NHS Test & Trace in the 'Personal Health' category.
